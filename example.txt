Справка по запуску:

При первом запуске необходимо установить requirements.txt для корректной работы:
>> pip install -r requirements.txt

Чтобы запустить сервер на MacOS или Linux необходимо ввести: 
>> sudo python3 dns_server.py 
после этого будет запрошен пароль от компьтера, так как использование порта с номером <80 требует разрешения пользователя

После этого на консоли появится сообщение о включении сервера: 
>> server works
>> port: 53

Далее необходимо воспользоваться клиентом для того, чтобы видеть информацию в удобном виде:
>> python3 begin.py DOMAIN TYPE (например, python3 begin.py e1.ru A)
В качестве типа записи вы можете использовать A, MX и TXT.
